## 规范和实现  
| 名称             | 层次             | 说明                                |
| -------------- | -------------- | --------------------------------- |
| 方法区            | JVM规范          | JVM必须要有的逻辑内存区域，定义存放什么内容，但没细说实现细节  |
| 永久代（PermGen）   | HotSpot JVM 实现 | Java 7 及以前实现方法区的具体内存结构，固定大小，堆内存区域 |
| 元空间（Metaspace） | HotSpot JVM 实现 | Java 8+ 实现方法区的具体内存结构，使用本地内存，动态可扩展 |
