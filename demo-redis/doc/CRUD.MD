## REDIS 支持 字符串 有序集合 无序集合 列表（阻塞也支持） hash（也叫做字典或者Map）等常用的，类型一般说的是值
### 键 ，redis的键都是字符串，键是全局的，整个redis 的键唯一，不分类型 
### 类型转换
| 类型     | 支持数值操作？ | 原因说明                 |
| ------ | ------- | -------------------- |
| String | ✅       | 值是字符串，支持 INCR、DECR 等 |
| Hash   | ✅       | 某个 field 的值支持数值加减    |
| ZSet   | ✅       | 成员的 score 支持加减       |
| Set    | ❌       | 无值结构，只有字符串元素         |
| List   | ❌       | 操作是基于位置，没有字段或 score  |
